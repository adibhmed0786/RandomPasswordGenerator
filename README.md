# Random Password Generator
A lightweight Python console app that generates secure, random passwords based on user-defined criteria, including length and inclusion of letters, numbers, and symbols. Built using Python's random and string libraries, it features input validation for reliable use. Perfect for beginners or anyone needing quick, customizable passwords.

# Features
- Customizable password length (minimum 4 characters).
- Optional inclusion of letters, numbers, and symbols.
- Input validation for length and character set selection.
- No external dependencies; uses standard Python libraries.

# Requirements
- Python 3.8+ (no external libraries required).

# Installation
1- Clone: git clone https://github.com/adibhmed0786/Random-Password-Generator.git

2- Navigate: cd Random-Password-Generator

# Usage
1- Run: python password_generator.py

2- Enter desired password length and select character sets (y/n).

3- Copy the generated password and store it securely.

# How It Works
- generate_password(): Creates a character pool based on user choices and generates a random password.
- main(): Collects user input, validates it, and displays the password.
